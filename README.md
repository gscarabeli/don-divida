# Don Dívida

**Seu Consultor Financeiro Sem Misericórdia**

Don Dívida é um chatbot de consultoria financeira pessoal que não poupa críticas. Inspirado no estilo de um mafioso sofisticado, o Don analisa seus números e dá conselhos diretos, sem filtros e sem misericórdia.

## Funcionalidades

- **Análise Financeira Personalizada**: Insira seus dados financeiros (renda, gastos, dívidas, etc.) para uma avaliação precisa.
- **Chat Interativo**: Converse com o Don sobre suas decisões financeiras. Ele responde com roast pesado e conselhos práticos.
- **Interface Minimalista**: Design escuro e elegante, focado na conversa.
- **Privacidade**: Sua chave de API e dados ficam apenas no seu navegador.

## Como Usar

1. **Clone ou baixe** este repositório.
2. **Abra o arquivo `index.html`** em qualquer navegador moderno.
3. **Configure sua API Key**:
   - Acesse [aistudio.google.com/apikey](https://aistudio.google.com/apikey).
   - Crie uma chave gratuita do Google Gemini.
   - Cole a chave no campo indicado.
4. **Insira seus dados financeiros**:
   - Nome (opcional)
   - Renda mensal
   - Gastos mensais
   - Saldo atual
   - Investimentos mensais
   - Total de dívidas
   - Reserva de emergência
5. **Clique em "O DON ESTÁ PRONTO PARA TE OUVIR"**.
6. **Converse!** Digite suas perguntas ou use os botões rápidos para tópicos comuns.

## Requisitos

- **Navegador**: Qualquer navegador moderno com suporte a ES6+ e Fetch API.
- **API Key**: Chave gratuita do Google Gemini (Gemini 2.0 Flash).
- **Conexão com Internet**: Necessária para as chamadas à API do Gemini.

## Tecnologias

- **HTML5**: Estrutura da página.
- **CSS3**: Estilização com variáveis CSS e animações.
- **JavaScript (Vanilla)**: Lógica do chat e integração com API.
- **Google Gemini API**: IA para gerar respostas do Don.

## Personalidade do Don

O Don fala como o Don Corleone das finanças:
- **Roast Pesado**: Críticas impiedosas a decisões ruins.
- **Direto e Grave**: Sem rodeios, com peso nas palavras.
- **Baseado em Números**: Sempre cita valores específicos do usuário.
- **Máximo 3 Parágrafos**: Respostas concisas e impactantes.

## Exemplos de Conversa

**Usuário:** Quero comprar um curso de R$800 de marketing digital.

**Don:** Comprar um curso de R$800 com R$ 1.500 de saldo e R$ 4.500 de dívida. Interessante estratégia. Você está construindo um castelo em cima de areia movediça e ainda quer colocar uma piscina.

## Limitações

- Funciona apenas offline após a configuração inicial (precisa de internet para API).
- Dados armazenados localmente no navegador (sessionStorage).
- Respostas limitadas a 800 tokens para manter o foco.
